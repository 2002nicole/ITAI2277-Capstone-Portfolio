# Important Resources Used

- DrugBank: biomedical drug and drug-drug interaction reference data.
- ChEMBL: bioactive molecule database used by the group for molecular data context.
- SIDER: side effect resource reviewed in the broader group project.
- STITCH: protein-chemical interaction data reviewed in the broader group project.
- RDKit: molecular informatics toolkit for SMILES and molecular feature concepts.
- pandas: tabular data processing in Python.
- NumPy: numerical computing in Python.
- scikit-learn: train/validation/test splitting and model utility functions.
- Google Colab: notebook environment used for coursework and capstone development.
- GitHub: repository organization, documentation, and portfolio presentation.
