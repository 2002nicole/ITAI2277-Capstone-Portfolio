# Nicole Marcial - AI Applications Portfolio (ITAI 2277)

This repository is an individual portfolio of selected work from ITAI 2277: Artificial Intelligence Resource Capstone. It is designed for employers, instructors, and collaborators who want a clear view of my applied AI skills, especially in data preprocessing, feature engineering, model pipeline support, and responsible AI documentation.

## Portfolio Focus

My strongest project contribution this semester was the DrugBank data engineering component for **CompoundIQ**, a group capstone project focused on AI-powered safe 3-drug combination discovery.

CompoundIQ was a **team project**. I am not presenting the full system as individual work. My individual contribution focused on preparing the DrugBank data used to support downstream drug-interaction modeling as well as presentations.

The full group repository is available here:

https://github.com/ComfortCS50/CompoundIQ

Please note: the full repository includes contributions from multiple team members.

## My Main Contributions

- Built the DrugBank data preprocessing and feature engineering workflow.
- Parsed raw biomedical DrugBank XML into structured, reusable tables.
- Created drug interaction, CYP enzyme, metabolic complexity, and safety complexity features.
- Prepared active drug subsets and train/validation/test splits.
- Completed and organized the Phase 2 data pipeline and EDA presentation.
- Reviewed related team notebooks, especially the EDA notebook, to understand how DrugBank, ChEMBL, SIDER, and STITCH connected in the full project pipeline.
- Created and organized the group GitHub repository for the CompoundIQ capstone project.
