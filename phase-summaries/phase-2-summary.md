# Phase 2 Summary - Data Pipeline and Exploratory Analysis

Phase 2 was my strongest individual contribution. I worked on the DrugBank data pipeline by parsing raw biomedical data, cleaning records, creating structured tables, engineering features, and explaining the EDA results.

Important DrugBank outputs included drug metadata, drug-drug interactions, targets, enzymes, interaction count features, CYP-related features, safety complexity scores, an active drug subset, and train/validation/test splits.

The Phase 2 presentation is included in this portfolio because I completed/organized that presentation and it directly represents my DrugBank work.
