# Phase 1 Summary - Project Proposal and Scope

CompoundIQ began as a group capstone proposal for an AI system that could help evaluate safe drug combinations. The original idea connected natural language mechanism input, molecular candidate generation, drug interaction prediction, and safety ranking.

My role at this stage was to understand the project direction and prepare for the DrugBank data work that would support the safety and interaction side of the pipeline.

**Portfolio note:** This was group planning work, so this repository summarizes the phase instead of presenting it as individual-only work.
