# Phase 3 Summary - Model Architecture, Training, and Evaluation

Phase 3 connected each team member's component into the larger model architecture. The full group system included BioBERT for mechanism text embeddings, JT-VAE for molecular generation, DrugBank features for safety and interaction context, and a 3-Way GNN for triplet interaction prediction.

My contribution was explaining how the DrugBank feature table supported the GNN pipeline. I focused on how engineered features such as CYP overlap, interaction degree, graph centrality, and safety complexity could help the model reason about drug interaction risk.
